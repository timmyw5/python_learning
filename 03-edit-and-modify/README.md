# ✏️ 板块三：编辑改造课

> 从"读"到"改"——在已有代码上动手修改和扩展功能。

## 学习目标
- 能安全地修改已有代码而不破坏原有功能
- 能给现有功能添加新特性
- 能重构代码使其更清晰
- 理解"改一处可能影响多处"的概念
