# 📗 板块二：阅读理解课

> 从"认字"到"读文章"——读懂完整的 Python 程序在做什么。

## 学习目标
- 能追踪程序的执行流程（代码是按什么顺序跑的）
- 能理解函数之间的调用关系
- 能复述一段程序的"大意"
- 能识别常见的代码模式和惯用写法
