# 🐍 Python 识读课 — AI时代的Python学习之路

> **不是从零写代码，而是学会读懂、编辑、和驾驭已有的Python代码。**

## 🎯 课程目标

在AI可以写出一切代码的时代，人类需要的不是敲键盘的速度，而是：

- **认识** Python 的语法元素，看到代码不再发慌
- **阅读** 完整的 Python 程序，理解它在做什么
- **编辑** 已有代码，修改和扩展功能
- **诊断** Bug，定位问题并修复
- **理解** Python 如何被运用于万物
- **协作** 与 AI 搭档高效完成真实任务

## 🎮 学习机制

### Duolingo 风格碎片化学习
- ⏱️ 没有固定时间要求，学完一节算一节
- 📖 每节课 = 知识讲解 + 代码示例 + 总结卡片 + 小测验
- 🔄 间隔复习：每2-3节新课插入回顾挑战
- 🎯 弱项追踪：未通过的知识点会被标记并强化

### 每节课结构
```
📖 知识讲解（带真实代码示例）
   ↓
🔍 代码阅读练习
   ↓
📋 总结卡片（3-5个要点）
   ↓
🧪 小测验（2-4题，混合题型）
   ↓
✅ 通过 → 下一节 / ❌ 未通过 → 复习补强
```

### 测验类型
- 📖 **阅读理解**：这段代码的输出是什么？
- 🐛 **找Bug**：这段代码有什么问题？
- ✏️ **改代码**：按要求修改 .py 文件
- 🧩 **概念匹配**：连线/填空
