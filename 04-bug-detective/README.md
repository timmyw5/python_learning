# 🐛 板块四：Bug侦探课

> 代码出了问题？学会像侦探一样追踪线索、定位真凶。

## 学习目标
- 能读懂 Python 的错误信息（Traceback）
- 能区分语法错误、运行时错误、逻辑错误
- 掌握基本的调试策略
- 能在复杂代码中定位问题所在
