# 📖 板块一：Python识字课

> 看到代码不慌，认识基本语法元素，建立代码直觉。

## 学习目标
- 认识 Python 的基本"词汇"：变量、数据类型、运算符
- 理解代码的基本"句法"：函数、条件、循环
- 建立"代码直觉"：看到一段代码能大概猜到它在做什么
